# 🏥 Hospital Management System

A Django-based web application for managing doctors, departments, and appointment bookings.

---

## 🚀 Features

* Doctor listing
* Department management
* Appointment booking
* Responsive UI (Bootstrap)

---

## 🛠️ Tech Stack

* Python
* Django
* HTML, CSS, Bootstrap
* SQLite

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
